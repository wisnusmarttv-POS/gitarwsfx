### Prerequisites

*   Node.js (v14 or higher)
*   npm

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/jagatraya2508/gtrfx.git
    cd gtrfx
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Run the development server:
    ```bash
    npm run dev
    ```

4.  Open `http://localhost:5173` in your browser.
